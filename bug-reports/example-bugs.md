# Example Bug Reports

## BUG-001 Logout option is not immediately visible after opening the sidebar

Environment:
- Windows 11
- Chrome
- Sauce Demo

Steps to reproduce:
1. Login with valid credentials
2. Open the burger menu

Expected result:
Logout option is immediately visible.

Actual result:
There can be a short delay before the logout option appears.

## BUG-002 Cart icon feedback is subtle after item is added

Environment:
- Windows 11
- Chrome
- Sauce Demo

Steps to reproduce:
1. Login with valid credentials
2. Add one item to the cart

Expected result:
Clear visual confirmation that the item was added.

Actual result:
The cart badge changes, but the feedback may be easy to miss.
